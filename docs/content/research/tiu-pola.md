## TIU Verbal

### Sinonim — kata persamaan

Pola SKD CPNS:
- Kata serapan / kata baku yang mungkin asing
- Sering muncul: PROAKTIF, DIKOTOMI, EFEKTIF, REKONSILIASI, MAYAPADA, INTRODUKSI, REPRESIF, EKSPLISIT, IMPLISIT, KOMPETENSI, DOMINASI, DISEMINASI, AGITASI, AKSELERASI, VERIFIKASI, MARGINAL

### Antonim — kata lawan

Pola:
- Pasangan klasik: KOLEKTIF×INDIVIDUAL, EKSPLISIT×IMPLISIT, ABSTRAK×KONKRET
- Sering muncul: MAKAR×SETIA, GENERIK×SPESIFIK, INTRODUKSI×PENUTUP, ANTAGONIS×PROTAGONIS, RIGID×FLEKSIBEL

### Analogi verbal — A:B = C:?

Pola relasi:
1. **Profesi-alat**: DOKTER:STETOSKOP = PETANI:CANGKUL
2. **Anggota-kelompok**: AYAM:UNGGAS = HIU:IKAN
3. **Sebab-akibat**: HUJAN:BANJIR = KEMARAU:KEKERINGAN
4. **Bagian-keseluruhan**: RODA:MOBIL = LENSA:KAMERA
5. **Fungsi-objek**: GUNTING:MEMOTONG = SAPU:MEMBERSIHKAN
6. **Tingkat**: HANGAT:PANAS = DINGIN:BEKU

### Silogisme

Pola SKD:
1. **Premis universal-partikular**: "Semua A adalah B. Sebagian B adalah C. Maka..."
2. **Premis dua kondisional**: "Jika A maka B. Jika B maka C. Maka jika A..."
3. **Modus tollens**: "Jika A maka B. Tidak B. Maka tidak A."
4. **Modus ponens**: "Jika A maka B. A. Maka B."

Aturan dasar:
- "Semua + sebagian = sebagian" (kesimpulan partikular)
- "Tidak ada A yang B = tidak ada B yang A" (konversi)
- Kesalahan umum: menyimpulkan dari "sebagian + sebagian" (tidak valid)

## TIU Numerik

### Deret bilangan

Pola umum:
- Aritmatika: U_n = a + (n-1)d
- Geometri: U_n = a × r^(n-1)
- Kombinasi: tambah → kuadrat → tambah
- Selang-seling: pola ganjil dan genap berbeda
- Fibonacci: U_n = U_(n-1) + U_(n-2)

### Hitung dasar

- Persen, perbandingan, pecahan
- KPK, FPB
- Rata-rata, rata-rata gabungan
- Deret aritmatika & geometri (jumlah n suku)

### Soal cerita umum

1. **Usia** — perbandingan usia sekarang vs dulu/akan datang
2. **Jarak-Waktu-Kecepatan** — searah, berlawanan, susul-menyusul
3. **Untung-Rugi-Diskon** — persentase, harga jual/beli
4. **Pekerja** — orang × hari × jam
5. **Campuran** — komposisi, kadar
6. **Bunga** — bunga tunggal, bunga majemuk

### Aljabar

- Persamaan linear satu/dua variabel
- Pertidaksamaan
- Sistem persamaan

## TIU Figural

### Tipe soal:

1. **Analogi gambar**: A:B seperti C:?
2. **Seri gambar**: pola yang berlanjut
3. **Ketidaksamaan**: 1 dari 5 berbeda
4. **Rotasi/refleksi**: putar/balik

### Pola visual umum:

- Rotasi 90°/180°/270° CW atau CCW
- Refleksi horizontal/vertikal
- Penambahan/penghilangan elemen
- Perubahan warna/arsiran (hitam ↔ putih)
- Perubahan jumlah elemen (1, 2, 3, 4)
- Perubahan posisi (kiri-tengah-kanan)

### Image prompt template

```
Layout: [grid 2x2 / row of 5 / single]
Style: line art, black on white, simple geometric shapes
Element: [circle / square / triangle / arrow / cross / dot]
Quantity: [number]
Position: [center / corner / edge]
Transform: [rotation X°, reflection axis, scale]
Note: aspect ratio 1:1 per cell, no text labels
```
